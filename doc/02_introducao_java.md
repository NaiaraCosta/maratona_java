# 📘 Guia Completo — Ambiente de Desenvolvimento Java (IDE IntelliJ IDEA)

Este guia apresenta como configurar um ambiente profissional de desenvolvimento Java utilizando uma IDE.

---

## 🧠 Resumo

O desenvolvimento Java pode ser feito manualmente, mas isso gera baixa produtividade.

Por isso utilizamos uma IDE (Integrated Development Environment), que automatiza tarefas como:

* Escrita de código
* Compilação automática
* Execução do programa
* Detecção de erros em tempo real

---

## 💻 IDE (Ambiente de Desenvolvimento)

Uma IDE é uma ferramenta que integra tudo que o desenvolvedor precisa.

## Principais recursos:

* Editor inteligente
* Sugestões automáticas de código
* Compilação integrada
* Execução com um clique
* Debugger

---

## 🔥 IntelliJ IDEA

O IntelliJ IDEA é uma das IDEs mais utilizadas para Java.

### Características:

* Alta produtividade
* Suporte completo ao Java
* Integração automática com JDK
* Muito usado no mercado internacional

### Versões:

* Community (gratuita)
* Ultimate (paga)

👉 Para estudos, a Community é suficiente.

---

### ⚙️ Configuração inicial

Após instalar:

* Selecionar JDK instalado
* Ajustar fonte e tema
* Instalar plugins opcionais

---

### 📌 Plugins úteis:

* Presentation Assistant (mostra atalhos na tela)

---

## 🚀 Resumo final

* IDE substitui trabalho manual
* IntelliJ automatiza o processo de desenvolvimento
* O foco passa a ser lógica e não configuração
