##### **➗✖️➖➕ Operadores**

- Aritméticos: +, -, *, /, %
- Comparação: ==, !=, <, >, <=, >=
- Lógicos: &&, ||, !
- Atribuição: =, +=, -=, *=, /=
- Incremento/Decremento: ++, --

----------------------------------------------------------------------------
count++ (pós-incremento): primeiro usa o valor atual, depois incrementa.

int count = 5;
int x = count++;

System.out.println(x); // 5
System.out.println(count); // 6

-----------------------------------
++count (pré-incremento): primeiro incrementa, depois usa o valor.

int count = 5;
int x = ++count;

System.out.println(x); // 6
System.out.println(count); // 6
----------------------------------------------------------------------------

Exemplos:

int a = 10;

int b = 5;

**System.out..println(a +b); // 15**

**System.out.println(a >b); // true**
