# 📘 Guia Completo — Palavras Reservadas (Keywords) em Java

Este guia apresenta o conceito de **keywords (palavras reservadas)** no Java, sua função na linguagem e como elas estruturam o comportamento do compilador.

---

## 🧠 O que são Keywords

Keywords são palavras reservadas da linguagem Java que possuem significado fixo para o compilador.

Elas fazem parte da estrutura base da linguagem e não podem ser alteradas ou redefinidas pelo desenvolvedor.

---

### ⚠️ Regra fundamental

Keywords **não podem ser usadas como identificadores**, ou seja:

* Nomes de variáveis
* Nomes de métodos
* Nomes de classes
* Nomes de pacotes

---

## 📊 Quantidade de Keywords

O Java possui aproximadamente **50+ keywords**, podendo variar de acordo com a versão da linguagem.

---

## 📦 Para que servem as Keywords

As keywords são responsáveis por definir a estrutura da linguagem Java.

Elas controlam:

* 🔄 Fluxo de execução
* 📊 Tipos de dados
* 🔐 Controle de acesso
* 🧱 Estrutura de orientação a objetos
* ⚙️ Comportamento do compilador

## ⚡ Regras importantes

* Keywords são **case-sensitive**
* Não podem ser redefinidas
* Não podem ser usadas como identificadores

---

## 🚨 Erros comuns

### ❌ Usar keyword como variável

```java
    int class = 10;
```

### ❌ Usar keyword como método

```java
    public void return() {}
```
